Important FAQ

What is the Document Object Model(DOM)?
Document Object Model(DOM) refers to a programming interface by which scripts are allowed to access and update the content, structure, and style of a web document. It is done most commonly with JavaScript.
How do you select an element from the DOM?
There are several methods in javascript to select an element from the DOM. Those methods are: Using a Unique ID, Using CSS Selectors, and Using Class or Tag Names.
What is event delegation in the context of the DOM, and why is it useful?
Event delegation is the way of attaching a single event listener to a parent element instead of attaching multiple listeners to its individual child elements. Events that "bubble up" from the children are managed by the single listener.
How do you manipulates an element's attribute and styles using the DOM?
Element's attributes and styles are manipulated by first selecting the element and then using specific properties and methods to change its HTML attributes or CSS styles. Attributes are values like id, class, src, or href inside an HTML tag. We can get, set, or remove them using dedicated methods.
